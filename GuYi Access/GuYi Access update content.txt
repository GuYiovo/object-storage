GuYi Access 更新日志
核心 API 矩阵全面升级，商业生态完美闭环！
本次更新对网络验证的核心业务接口进行了深度扩容，大幅提升了系统的商业对接能力与安全性：

增设高危主动防御接口： 开放 ban_machine 接口，支持客户端检测到破解（如OD/CE附加、抓包）时，自动将黑客设备与 IP 永久拉入云端黑名单！
开放发卡网/管理端 API 对接： 新增专属授权的 api_token 机制，完美支持第三方发卡网自动提卡发货、机器人自助管理。
强化在线管理能力： 新增服务端“无损踢下线”及后台封解禁接口，远程管控设备更加灵活。
云端配置动态下发： 验证接口完美支持读写 custom_data（云自定义配置），支持免登拉取软件最新版本号及全局公告。
